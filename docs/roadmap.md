# Roadmap

This is the public-facing roadmap summary for AITP.

## Direction

AITP is still moving toward a protocol-first research runtime where:

- charter and contract surfaces remain public and reviewable
- the runtime stays narrow and auditable
- agent adapters act as protocol executors rather than hidden workflow brains

## Public reading rule

This page is intentionally public-facing.
It should explain direction and stable public documents without requiring
operators to read internal planning state.

maintainer-facing planning state lives in internal workflow artifacts under the
repository root and is not part of the public roadmap surface.

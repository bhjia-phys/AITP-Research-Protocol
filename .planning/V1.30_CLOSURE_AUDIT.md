## Physicist-Facing Judgment

From the perspective of a theoretical physicist using AITP:

1. AITP now remembers topic-derived route knowledge instead of only a fixed
   demo graph.
2. When that memory matters, the system can surface what was retrieved, why it
   was surfaced, which warnings apply, and where to read next.
3. The human is no longer forced into a fake checkpoint for every meaningful
   memory/result change; `non_blocking_update` now exists for the common case
   where new information matters but no route-changing answer is required.

This is materially better than the prior state.

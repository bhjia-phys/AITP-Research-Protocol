# Requirements

Milestone: `v1.36` `L0 L1 Source Intelligence And Physicist-Grade Intake`

## Requirement Set

- `REQ-SRC-01` AITP must assign more durable source identity so the same paper
  or note can be recognized across related topics.
- `REQ-SRC-02` AITP must surface citation-graph and source-neighbor signals more
  explicitly than the current minimal baseline.
- `REQ-L1-01` `L1` intake must carry stronger assumption, regime, and reading
  depth structure.
- `REQ-L1-02` `L1` intake must begin surfacing contradiction candidates or
  notation-alignment tension instead of only shallow extraction.
- `REQ-L1-03` The stronger `L0/L1` intelligence must appear in runtime/topic
  read paths so later layers can actually use it.

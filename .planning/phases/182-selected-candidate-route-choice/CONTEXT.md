# Phase 182 Context: Selected-Candidate Route Choice

## Why this phase exists

After `v2.7`, the same fresh topic can already execute consultation-followup
and land on one selected staged candidate. The next bounded gap is that the
loop still stalls on the candidate-specific summary instead of deriving the
first deeper route choice.

## Bounded goal

Derive one bounded route choice from the selected staged candidate so the loop
can move beyond candidate-selection summary into either split, validate, or
promotion review posture.

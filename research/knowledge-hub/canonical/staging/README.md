# L2 staging

This folder is the provisional entry surface for reusable `L2` candidates that
are worth remembering but are not yet trusted canonical memory.

Use staging for:

- reusable discussion insights,
- bounded physical pictures or heuristics,
- warning candidates,
- lightweight route notes,
- and other low-friction captures that should remain durable before review.

Do not confuse staging with canonical `L2`.

- staging is provisional
- canonical is authoritative

The source-of-truth index is:

- `staging_index.jsonl`

Entry records must conform to:

- `../../schemas/l2-staging-entry.schema.json`
